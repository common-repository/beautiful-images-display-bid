Beautiful Image Display (BID) plugin is a lightweight and responsive WordPress plugin capable of displaying images in a beautiful overlay

== Description ==

IWF1’s Beautiful Image Display (BID) plugin is a lightweight and responsive – fits all relevant screen sizes – WordPress plugin.

BID plugin supports both mouse and keyboard bindings to manipulate images display:

* Click the X button or hit Esc key to close the display.
* Navigate multiple post images by clicking on the area of the displayed arrows or by pressing your keyboard arrow keys (left and right).

Currently supported image formats are: **JPG, JPEG, PNG, GIF** and **SVG**. Other formates may be added according to demand.

Embedded YouTube videos are now partly supported as well.

BID is geared towards performance, written entirely in pure Javascript – i.e no extra libraries needed / loaded by the plugin such as jQuery and others.

The plugin Supports RTL languages as well.

== Installation ==

= Auto install via WordPress =

1. Log into WordPress as admin user. 
2. Go into plugins and click "Add New".
3. Search for BID (or Beautiful Image Display).
4. Click "Install Now" to install it.
5. Go into plugins screen and activate the plugin.

= Manual =

1. Download BID to your desktop.
2. If downloaded as a zip archive, extract the Plugin folder to your desktop.
3. With your FTP program, upload the Plugin folder to the wp-content/plugins folder in your WordPress directory online.
4. Go to Plugins screen and find the newly uploaded Plugin in the list.
5. Click Activate to activate it.



== Frequently Asked Questions ==

= Is the plugin configurable? =

At the moment, there's an option to configure buttons colors (i.e close and arrow buttons).

= What about other formats? =

As mentioned in the description, more formats can and will be added according to demand.
